## ⚙️ Setup

### 🔧 Prerequisites
- Node.js 20+  
- Python 3.11+  

---

### ▶️ Frontend

```bash
cd frontend
npm install
npm run dev

▶️ Backend
cd backend
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/macOS
source venv/bin/activate

pip install -r requirements.txt
uvicorn main:app --reload
```

### 🎮 Usage

Try commands like:

"Change background to black"
"Add a button"
"Increase font size"
"Create a navbar"

```bash
🧩 Example Command
{
  "action": "update_style",
  "target": "body",
  "property": "background-color",
  "value": "#0000ff"
}
```

📦 Build
cd frontend
npm run build
