Place your AURA 3D model file here:

- Required path: `frontend/public/models/aura.glb`
- Supported by current loader: `.glb`

Notes:
- Keep model centered around origin if possible.
- If the model scale is too large/small, adjust `scale` in `AuraRobot.jsx`.
- While the model is loading (or if missing), VoxForge shows a procedural fallback robot.
